# CarOS

A web-based car operating system interface built with HTML, CSS, and JavaScript.

## About

CarOS is a personal project that I created as an interactive OS interface focused on cars. 
The goal was to create a desktop-like interface that can show and display my love of cars.

## Features

- 🖥️ OS-style interface
- 🚘 Car showcase
- 🔧 Custom apps
- 🎨 Custom UI design
- 📱 Responsive layout
- 📱 Embedded links

## Technologies Used

- HTML
- CSS
- JavaScript
