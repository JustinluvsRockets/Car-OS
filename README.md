# CarOS

A web-based car operating system interface built with HTML, CSS, and JavaScript.

## About

CarOS is a personal project that creates an interactive OS interface focused on cars. 
The goal was to create a desktop-like interface that can show and display my love of cars.

## Features

- 🖥️ OS-style interface
- 🚘 Car showcase
- 🔧 Custom apps and tools
- 🎨 Custom UI design
- 📱 Responsive layout

## Technologies Used

- HTML
- CSS
- JavaScript
